# Amazon Revenue Prediction

A machine learning project exploring how demographic and purchasing behavior data can be used to predict monthly Amazon revenue.

## Motivation

I wanted to compare several regression and ensemble learning techniques on a real-world dataset while improving my understanding of feature engineering, model evaluation, and predictive performance.

## Methods

- Linear Regression
- Linear Regression with Interaction Terms
- Random Forest
- XGBoost

## Tools

- R
- tidymodels
- ranger
- xgboost
- ggplot2

## Result

Random Forest achieved the strongest cross-validated predictive performance among the candidate models.
